Please place the following images in this folder:

1. **NYHSSfavicon.png** - The atom icon (used as favicon for all pages)
2. **NYHSSlogo.png** - The full logo with text (used in header)

- **NYHSSfavicon.png**: Should be the atom icon, recommended size 32x32 or 64x64 pixels
- **NYHSSlogo.png**: The full horizontal logo with "NEW YORK HUNGARIAN SCIENTIFIC SOCIETY" text
- **hero-image.png**: The dual-city skyline illustration showing NYC and Budapest
